Stack Machine Simulator

  • A Stack Machine is a real or emulated computer which uses pushdown stack to evaluate each sub-expression in the program. It is programmed with a reverse Polish notation instruction set.
  • A Java Program (using Java Swing), simulating the interpretation of code written in assembly language having instructions: push, pop, add, sub, multiply, div, display. It reads program line by line and displays fetch and execution cycle.
  • It works on concept of stack pointer. For push the pointer value must be atleast two and for pop atleast one. For pop operation, it decreases stack pointer by one popping out value. In push operation it take two items performs operation and put result back in stack thus increasing stack pointer by one.
